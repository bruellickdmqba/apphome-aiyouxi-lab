# apphome-aiyouxi-lab

本仓库用于归档和发布多个独立的 HTML 页面。

## 项目简介

这是一个用于存放 HTML 页面资源的仓库。仓库中的页面彼此独立，不依赖特定后端服务，可直接在浏览器中打开运行。页面内容涵盖工具、展示、实验性交互等多种类型。

## 目录说明

```
.
├── pages/          # 存放各独立 HTML 页面及附属资源
│   └── ...
├── assets/         # 公共资源文件（样式、脚本、图片等）
│   └── ...
└── README.md       # 本文件
```

- `pages/` 目录下每个子文件夹对应一个独立的页面，包含该页面的 HTML、CSS、JavaScript 文件及相关资源。
- `assets/` 目录存放多个页面可能共用的资源文件，如基础样式库、通用工具脚本等。

## 页面归档说明

- 每个页面均以独立文件夹形式归档，命名尽量体现页面用途或主题。
- 页面遵循 HTML5 标准，尽量兼容主流现代浏览器。
- 部分页面可能包含实验性功能，不保证在所有环境下稳定运行。

## 使用方式

1. 克隆或下载本仓库。
2. 在 `pages/` 目录下找到感兴趣的页面文件夹。
3. 直接双击打开其中的 `index.html` 文件，或通过本地服务器运行。

## 维护说明

- 本仓库不定期更新，新增或修改页面会通过 commit 记录说明。
- 欢迎提交 Issue 反馈问题或建议。
- 不接受直接向仓库推送修改的 Pull Request，如有改进想法，请先通过 Issue 讨论。
